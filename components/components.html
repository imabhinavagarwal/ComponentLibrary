Avatar

The Avatar component is used to represent a user, displays the profile picture, initials or fallback icon.

Round Avatar

Easy Style has three circle shaped avatar which includes avatar large, avatar and avatar small. Size of Round Avatar is created by passing class="avatar round" with l or sm for large and small respectively. When nothing is passed it will load default as medium

Square Avatar

Easy Style has three square shaped avatar which includes avatar large, avatar and avatar small. Size of Sqaure Avatar is created by passing class="avatar" with l or sm for large and small respectively. When nothing is passed it will load default as medium

Badge

Badges are the small circle components which is generally used at the parent component to display numbers. It can be situated at the top-right, top-left or at the bottom of the parent component.

Numbered Badge

Numbered badges are used to get the notification count on the parent container.

15
10

Dot Badge

Dot badges are used to show the dot on the parent container. This is widely used to show unread notifications or online/offline status
