
# Easy UI

# Easy Style
It's a CSS Library with which you can design and build some of your awesome projects as Easy Style

has some pre-defined styled classes and components.

## Features

- Avatar

  - Rounded Avatar
  - Square Avatar

- Alert

  - Simple Alert
  - Outline Alert

- Badge

  - Numbered Badge
  - Dot Badge

- Card
	- Basic Card
	- Shadow Card
	
- Check Box

- Progess Bar

- Snackbar


  - Baseline Snackbar

- Slider

- Buttons
  - Primary Buttons
  - Rounded Buttons
  - Buttons with Link

- Slider

- Collapsible
