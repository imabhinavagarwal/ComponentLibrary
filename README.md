# Easy UI

It's a CSS Library with which you can design and build some of your awesome projects as Easy UI
has some pre-defined styled classes and components.

## Features

- Avatar

  - Rounded Avatar
  - Square Avatar

- Alert

  - Simple Alert
  - Outline Alert

- Badge

  - Numbered Badge
  - Dot Badge

- Snackbar

  - Baseline Snackbar

- Slider

- Buttons
  - Primary Buttons
  - Rounded Buttons
  - Buttons with Link
